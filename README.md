# ⚛️ Quantum Computing with Qiskit

Exploring **Quantum Computing** fundamentals using IBM's Qiskit framework — quantum circuits, gates, entanglement, and algorithms.

---

## 📌 About This Project

As an ECE student, quantum computing felt like a natural extension of my studies — it blends physics, mathematics, and computing. This project covers the fundamentals of quantum circuits and demonstrates real quantum phenomena.

---

## 🔬 Topics Covered

| Concept | Description |
|---------|-------------|
| **Superposition** | Qubit existing in |0⟩ and |1⟩ simultaneously using Hadamard gate |
| **Entanglement** | Bell state — two perfectly correlated qubits |
| **Quantum Gates** | H gate, CNOT, X gate, Pauli gates |
| **Deutsch Algorithm** | Quantum speedup — 1 query vs 2 classical queries |
| **Bloch Sphere** | Visual representation of qubit state |

## ⚙️ Setup

```bash
pip install qiskit qiskit-aer matplotlib numpy
python quantum_basics.py
```

> **Note:** The script works even without Qiskit — it shows simulated results so you can understand the concepts!
